# @bryntum/demo-resources

This package contains fonts, images and SCSS files used by Bryntum demos for styling of headers and providing the resources. It is not needed in the final end-user application, however, it can serve as a model or basis for the application actual resources.

## Installation
If you ever need it, the installation is performed by issuing the installation command in the root of the application folder.
The specific command depends on the package manager used by the application.
The following are most frequently used:

Install using `npm`:
```
npm install @bryntum/demo-resources
```
Install using `yarn`:
```
yarn add @bryntum/demo-resources
```
